# Team Task Manager App 💬

A full-stack team task management web application built using **Flask + MySQL**.  
It allows users to create projects, assign tasks, and track progress with role-based access control.

---

## 🚀 Features

- 🔐 Secure User Authentication (Signup/Login)
- 👥 Role-based Access Control (Admin / Member)
- 📋 Project & Task Management System
- 📊 Task Status Tracking (Pending / In Progress / Completed)
- ⏰ Overdue Task Monitoring
- 👤 User Dashboard with Assigned Tasks

---

## 🛠 Tech Stack

- **Backend:** Python (Flask)
- **Database:** MySQL
- **Frontend:** HTML, CSS, Bootstrap
- **Deployment:** Railway
- **Tools:** Git, GitHub

---

## 🔐 Security Features

- Password hashing using Werkzeug
- Session-based authentication
- Role-based access control (RBAC)
- Parameterized SQL queries (Prevents SQL Injection)
